# SafeShare 경쟁 서비스 분석 & 한국형 재설계

## 경쟁 서비스 분석

### 1. sharepdf.io
- PDF 업로드 → 커스텀 링크 생성 (무료)
- 심플한 UI, 별도 가입 불필요
- **우리가 가져올 것**: 간편한 업로드 → 즉시 링크 생성 플로우

### 2. safeshare.tv
- 교사가 YouTube 영상을 광고 없이 안전하게 공유
- 교육 현장 특화 (Google Classroom 연동, 재생 구간 설정)
- **우리가 가져올 것**: "교육자를 위한 안전한 도구"라는 포지셔닝, 신뢰감 있는 톤

### 3. heyzine.com
- PDF → 페이지 넘김 효과가 있는 플립북으로 변환
- QR코드 자동 생성, 공유 링크 커스터마이징
- 비밀번호 보호, 조회 통계
- **우리가 가져올 것**: QR코드 생성 (한국 교사들이 종이 가정통신문에 QR 삽입 多), 
  깔끔한 뷰어 UI, 향후 플립북 효과 고려

## 한국 시장 특화 전략

### 한국 교사/공무원의 문서 공유 패턴
1. 가정통신문 PDF → **카카오톡 학급 단체방**에 공유 (90% 이상)
2. 학교 홈페이지 게시판에 PDF 첨부 (접근성 떨어짐)
3. 종이 가정통신문에 **QR코드** 인쇄하여 배포 (증가 추세)
4. 네이버 밴드에 파일 공유
5. e알리미(학교알림장 앱)로 전달

### 한국형 차별점
1. **QR코드 자동 생성** — 링크 생성 시 QR코드도 함께 제공. 종이 통신문에 인쇄하여 사용
2. **카카오톡 공유 최적화** — OG 태그 한글화, 카카오톡 인앱브라우저 완벽 대응
3. **한글 UI 전체** — 영어 없이 완전한 한글 인터페이스
4. **모바일 완전 최적화** — 한국 스마트폰 보급률 97%, 학부모 대부분 모바일로 접근
5. **네이버 SEO** — "PDF 링크 공유", "가정통신문 공유" 등 키워드 최적화
6. **한글 폰트 최적화** — Pretendard로 깔끔한 한글 렌더링

## 50MB 제한 안내 전략

50MB 제한을 부정적으로 느끼지 않도록 자연스럽게 노출:

1. **드래그앤드롭 영역 하단** (항상 보임, 흐린 텍스트):
   `무료 · 최대 50MB · 회원가입 불필요`

2. **50MB 초과 시 에러 메시지** (친절한 톤):
   `📎 파일 크기가 50MB를 초과합니다.`
   `현재 무료 서비스로 50MB 이하 파일만 지원됩니다.`
   `PDF 용량을 줄이려면 iLovePDF 압축을 이용해보세요. →`

3. **FAQ에서 자연스럽게**:
   `▸ 파일 크기 제한이 있나요?`
   `→ 무료로 최대 50MB까지 업로드할 수 있습니다.`
   `   대부분의 교육자료는 이 범위 안에서 충분히 공유 가능합니다.`

## QR코드 기능 상세

### 생성 시점
- 링크 생성과 동시에 QR코드 자동 생성 (기본 활성화)
- 체크박스로 비활성화 가능 (불필요한 경우)

### QR코드 제공 형태
- 화면에 QR코드 이미지 바로 표시
- "QR코드 이미지 저장" 버튼 (PNG 다운로드)
- 인쇄용 고해상도 옵션 (300dpi)

### 사용 시나리오
```
선생님이 종이 가정통신문 하단에 QR코드를 삽입
→ 학부모가 스마트폰으로 QR코드 스캔
→ PDF 뷰어 페이지로 바로 이동
→ 디지털 버전으로 상세 내용 확인
```

### 기술 구현
- 라이브러리: qrcode (npm) — 클라이언트 사이드 생성
- 크기: 기본 200x200px, 인쇄용 600x600px
- 포맷: PNG (투명 배경 옵션)
- QR코드 하단에 서비스명 표시: "PDF 링크공유기"

## 카카오톡 공유 최적화

### OG 태그 (한글 최적화)
```html
<meta property="og:title" content="2학년_가정통신문.pdf" />
<meta property="og:description" content="📄 안전하게 공유된 문서입니다. 탭하여 열람하세요." />
<meta property="og:image" content="/og-preview.png" />
<meta property="og:type" content="website" />
```

### 카카오톡 인앱브라우저 대응
- navigator.clipboard fallback (execCommand)
- 터치 이벤트 최적화
- viewport meta 설정 (확대/축소 허용)
- position: fixed 이슈 대응 (iOS 인앱브라우저)

### 카카오톡 공유 API (선택사항, Phase 2)
- Kakao JavaScript SDK 연동
- 커스텀 공유 메시지 (썸네일 + 설명 + 버튼)
- "문서 열기" 버튼이 있는 카카오톡 메시지

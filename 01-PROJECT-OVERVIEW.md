# SafeShare - 안전한 문서 공유 서비스

## 프로젝트 개요

공무원(주로 교사)이 학부모 등에게 PDF 교육자료를 안전하게 공유할 수 있는 웹 서비스.
드래그앤드롭으로 PDF를 업로드하면 고유한 공유 링크가 생성되고, 설정한 기간이 지나면 자동으로 삭제된다.

## 핵심 가치

- **간편함**: 회원가입 없이, 드래그앤드롭 한 번으로 공유 링크 생성
- **안전함**: 기간 만료 자동 삭제, 보안 안내 문구, 다운로드 제한 옵션
- **신뢰감**: 공무원 대상 서비스답게 깔끔하고 신뢰감 있는 UI
- **무료**: 서버리스 아키텍처로 운영비 최소화

## 대상 사용자

- **작성자(업로더)**: 교사, 공무원 등 자료를 배포해야 하는 사람
- **열람자(뷰어)**: 학부모, 민원인 등 자료를 받아보는 사람 (모바일 위주)

## 핵심 플로우

```
[작성자]
1. safeshare 사이트 접속
2. PDF 파일 드래그앤드롭
3. 만료 기간 선택 (1일 / 7일 / 30일 / 90일)
4. (선택) 다운로드 허용 여부 설정
5. "공유 링크 생성" 클릭
6. 고유 링크 복사 → 카카오톡 등으로 전달

[열람자]
1. 카카오톡에서 링크 클릭
2. 모바일 브라우저에서 PDF 뷰어 열림
3. 보안 안내 문구 확인 (상단 배너)
4. 문서 열람 (핀치줌, 페이지 넘김)
5. (허용 시) 다운로드 버튼으로 저장
```

## 기술 스택

| 영역 | 기술 | 이유 |
|------|------|------|
| Frontend | Next.js 14+ (App Router) + TypeScript | SSR/SSG, API Routes 통합 |
| Styling | Tailwind CSS | 빠른 개발, 반응형 |
| PDF Viewer | @react-pdf-viewer/core (pdf.js 기반) | 모바일 지원, 커스터마이징 |
| Storage | Cloudflare R2 | S3 호환, egress 무료 |
| Database | Cloudflare D1 (SQLite) | 서버리스, 무료 티어 |
| Deployment | Cloudflare Pages | 무료, 글로벌 CDN |
| API | Next.js API Routes (Edge Runtime) | Cloudflare Workers 호환 |
| ID 생성 | nanoid | 짧고 안전한 고유 ID |

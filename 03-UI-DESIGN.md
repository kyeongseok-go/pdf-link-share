# SafeShare - 페이지 및 UI 설계

## 디자인 방향

### 톤 & 무드
- **신뢰감 있는 미니멀**: 공무원/교사 대상이므로 장난스럽거나 화려한 UI는 피한다
- **따뜻한 전문성**: 딱딱하지 않으면서도 안정감을 주는 디자인
- **모바일 퍼스트**: 열람자의 80%는 카카오톡 → 모바일 브라우저 경로

### 컬러 팔레트
```
Primary:    #1B4965  (Deep Navy - 신뢰, 안정)
Secondary:  #5FA8D3  (Sky Blue - 친근, 깨끗)
Accent:     #62B6CB  (Teal - 행동 유도)
Success:    #2D6A4F  (Forest Green)
Warning:    #E76F51  (Warm Coral - 만료 경고)
Background: #FAFBFC  (Off-white)
Surface:    #FFFFFF
Text:       #1A1A2E  (Nearly Black)
Muted:      #6B7280  (Gray)
```

### 타이포그래피
- 한글: Pretendard (가독성, 현대적)
- 영문: 동일 Pretendard 또는 Plus Jakarta Sans
- 모노스페이스 (링크 표시용): JetBrains Mono

### 인터랙션 상세

1. **드래그앤드롭 영역**
   - 기본: 점선 테두리, 회색 배경
   - 파일 드래그 진입 시: 테두리 색상 변경 (Primary), 배경 연한 블루
   - 파일 드롭 완료: 체크 아이콘 + 파일명 표시
   - PDF 이외 파일 시: 빨간 테두리 + "PDF 파일만 업로드 가능합니다" 에러
   - 50MB 초과 시: "파일 크기는 50MB 이하만 가능합니다" 에러

2. **업로드 진행**
   - "공유 링크 생성하기" 클릭 → 버튼이 프로그레스 바로 변환
   - 업로드 완료 → 스무스 트랜지션으로 결과 카드 표시

3. **링크 복사**
   - 클릭 → 클립보드 복사 + "복사 완료!" 토스트
   - 링크 텍스트 전체 선택 (모바일에서 수동 복사 대비)

## 페이지 2: 문서 뷰어 페이지 ( /v/[shareId] )

학부모가 보게 되는 페이지. 모바일 최적화가 최우선.

### 레이아웃

```
┌─────────────────────────────────────────────┐
│  🔒 SafeShare                               │  ← 최소한의 헤더
├─────────────────────────────────────────────┤
│  ┌──────────────────────────────────────┐   │
│  │  🔒 이 문서는 안전하게 공유된            │   │  ← 보안 안내 배너
│  │     자료입니다.                        │   │     (접을 수 있음)
│  │  • 공유 기간 종료 시 자동 삭제됩니다     │   │
│  │  • 본 문서는 열람 목적으로만 제공됩니다   │   │
│  │  📅 열람 가능 기간: ~2026.03.20        │   │
│  └──────────────────────────────────────┘   │
│                                             │
│  📄 2학년_가정통신문.pdf                     │  ← 파일명
│                                             │
│  ┌──────────────────────────────────────┐   │
│  │                                      │   │
│  │                                      │   │
│  │           PDF 뷰어 영역               │   │  ← pdf.js 뷰어
│  │          (핀치줌, 스크롤)              │   │
│  │                                      │   │
│  │                                      │   │
│  │                                      │   │
│  └──────────────────────────────────────┘   │
│                                             │
│  ◀ 1 / 3 ▶        🔍+ 🔍-    [ ⬇ 다운로드 ] │  ← 페이지 컨트롤 (하단 고정)
└─────────────────────────────────────────────┘
```

### 보안 안내 배너 상세

이 배너는 열람자에게 신뢰감을 주는 핵심 요소이다.

```
기본 문구 (상시 표시):
─────────────────────────────────
🔒 안전하게 공유된 문서입니다

이 문서는 [작성자]가 SafeShare를 통해
안전하게 공유한 자료입니다.

• 설정된 기간이 지나면 자동으로 삭제됩니다
• 외부에 무단 배포하지 말아 주세요
• 열람 기간: 2026.03.13 ~ 2026.03.20

SafeShare는 업로드된 파일을 열람 제공
외 어떤 용도로도 사용하지 않습니다.
─────────────────────────────────
```

### 만료 임박 시 문구 변경

```
만료 3일 이내:
─────────────────────────────────
⚠️ 이 문서의 열람 기간이 곧 만료됩니다

남은 기간: 2일 14시간
만료일: 2026.03.20

만료 후에는 문서에 접근할 수 없습니다.
필요하시면 지금 다운로드해 주세요.
─────────────────────────────────

만료 후:
─────────────────────────────────
🚫 문서 열람 기간이 만료되었습니다

이 문서는 공유 기간이 종료되어
자동으로 삭제되었습니다.

필요하시면 문서 작성자에게
다시 공유를 요청해 주세요.
─────────────────────────────────
```

### PDF 뷰어 요구사항

- 스크롤 기반 페이지 넘김 (모바일 자연스러움)
- 핀치 투 줌 (모바일 터치)
- 하단 툴바: 페이지 이동, 확대/축소, 다운로드 버튼
- 로딩 중: 스켈레톤 UI 또는 스피너
- 다운로드 비허용 시: 다운로드 버튼 숨김
- 다크모드 미지원 (인쇄 문서 특성상 밝은 배경 고정)

## 반응형 브레이크포인트

```

Tablet:   640-1024px
Desktop:  > 1024px  ← 업로더가 주로 사용
```

## 애니메이션 & 마이크로인터랙션

1. 파일 드롭 시 바운스 애니메이션
2. 링크 생성 시 결과 카드 슬라이드업
3. 링크 복사 시 체크마크 아이콘 전환 (0.3s)
4. 보안 배너 접기/펼치기 (accordion)
5. 업로드 프로그레스 바 (그라디언트 애니메이션)
6. 페이지 전환 시 페이드 트랜지션
